String Reversal

For each character being swapped, we are using Two Pointers approach. We set a pointer to the start of the array and a pointer at the end of the array.
During each iteration, for each character, we will swap start pointer character with end pointer character.
Once we're doing swapping the current characters, we increment start pointer and decrement end pointer to move onto the next characters until we're finished.
